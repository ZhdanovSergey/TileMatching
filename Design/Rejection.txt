Я пока не знаю, чем заменить отбракованные классы, но вот список классов, которые не проходят по каким-то критериям:

InputHandler
	этот класс должен только создавать объект Action из строки, введенной пользователем
	(инкапсулирует некоторое действие, функцию, а не задаёт структуру данных с оригинальным набором операций)

Renderer
	этот класс должен только выводить игровое состояние в консоль
	(инкапсулирует некоторое действие, функцию, а не задаёт структуру данных с оригинальным набором операций)

Action
	Это класс без методов, но ведь он реализует паттерн Command, в нем и не должно быть методов!
